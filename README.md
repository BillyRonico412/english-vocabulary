# ENGLISH VOCABULARY

## Objectif

Une application qui va faciliter de mémoriser les vocabulaires en Anglais

## Installation

* Clonner tout d'abord le depot

* Ajouter un fichier __config.json__ dans le dossier __assets__, et mettre dans celle-ci les champs suivants

    * port: Pour qui sera utilisé par l'application

    * routeAPI: Route qui sera utilisé par notre API, mettez '/api/vocabulaire'

    * motDePasse: Le mot de passe pour acceder à notre application

    * secertSession: Le hashage pour les informations stocké dans nos séssions

* Faire un coup de npm start
